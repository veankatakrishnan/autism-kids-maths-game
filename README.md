**Math Explorer** is a specialized, interactive educational web application designed to teach fundamental mathematical conceptsâ€”Rounding, Place Value, and Roman Numeralsâ€”to children, specifically tailored with neurodivergent (Autistic) learners in mind.

### 1. ğŸ¢ Rounding Roller Coaster (Rounding Game)
*   **Topic**: Rounding numbers to the nearest 10 or 100.
*   **Use Case**: Teaches spatial number sense.
*   **Mechanics**: Users are presented with a horizontal, interactive Number Line. They must visually determine which "major tick" (multiple of 10) the target number is closest to and input their answer.

### 2. ğŸ§± Place Value Builder (Place Value Game)

*   **Use Case**: Deconstructs abstract numbers into concrete, countable visual blocks.
*   **Mechanics**: Users must "build" a target number (e.g., `124`) by clicking `+` and `-` buttons to add distinct grid blocks into horizontal rows corresponding to Hundreds, Tens, and Ones.

### 3. ğŸ›ï¸ Roman Numeral Quest (Roman Numerals Game)

*   **Use Case**: Teaches symbol-to-value translation and addition/subtraction rules in numeral formatting.
*   **Mechanics**: Users construct the Roman Numeral string for a given integer using a dedicated, minimalist keypad.

## ğŸ“– Instructions for First-Time Users

Welcome to Math Explorer! Here is how to get started:

1.  **Open the App**: Navigate to the [Live Link](https://veankatakrishnan.github.io/autism-kids-maths-game/).
2.  **Choose a Module**: On the stark white Home Page, you will see three main game boxes. Click **"Play Now â†’"** on any game of your choice.
3.  **Playing the Game**:
    *   Read the target objective at the top of the screen (e.g., *"Round the number 59 to the nearest 10"*).
    *   Use the interactive elements (Number Line, Block +/- buttons, or Keypad) to form your answer.
    *   Click the solid black **Check Answer** button.
    *   You will receive highly contrasted, clear feedback (Success vs. Try Again).
4.  **Tracking Progress**: At any time, click the **Progress** button in the top navigation bar to view your overall score and accuracy charts.
5.  **Navigation**: Use the crisp white Navigation Bar at the top to seamlessly switch between games without losing your flow.

## ğŸ› ï¸ Developer Setup (Local Installation)

If you are a developer looking to run this application locally:

**Prerequisites:** Node.js (v16+) and npm installed.

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/veankatakrishnan/autism-kids-maths-game.git
    cd autism-kids-maths-game
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the local development server**:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173`.
4.  **Deploying Updates**:
    To push new updates to GitHub Pages, simply run:
    ```bash
    npm run deploy
    ```
